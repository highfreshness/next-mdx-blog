description: cmd.exe 실행 시 바로 꺼지는 현상 문제 해결
date: 2024-01-01
published: true
---

1. 현상
    1. Anaconda 설치 후 cmd Anaconda Prompt, cmder 실행 시 발 꺼짐
2. 확인
    1. 구글링을 통해 Anaconda 설치 후 간혹 이런 현상이 있는 경우가 있다는 사실을 알았고 처리 방법 역시 존재해 학인해 처리
    2. 처리
        1. PowerShell은 실행이 되기 때문에 PowerShell 에서 `C:\Windows\System32\reg.exe DELETE "HKCU\Software\Microsoft\Command Processor" /v AutoRun /f` 입력 후 실행
